Install Poetry

```bash
pip install poetry
```

Install dependencies

```bash
poetry install
```

Add dependencies:

```bash
poetry add click cwltool
```

Run using Poetry

```bash
poetry run d2s start virtuoso
```

Publish

https://codingdose.info/2019/06/16/develop-and-publish-with-poetry/

```bash
poetry build

# https://pypi.org/account/register/
poetry publish
```

Install the tool

```bash
pip install d2s
```
